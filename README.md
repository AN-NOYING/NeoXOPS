# NeoXOPS

[OpenXOPS](http://openxops.net/ "OpenXOPS")의 파생 프로젝트, NeoXOPS입니다.

Neo는 '새로운'이라는 뜻을 가지고 있으며, XOPS의 새로운 시작, 발전을 기원하며 붙인 이름입니다.

## Goals

- [ ] DirectX 11 &rightarrow; Vulkan (이건 엄청 오래 걸릴 듯)
- [ ] Joypad
- [ ] Networking
- [ ] glTF/glB (First Priority)
